# Product Grunt

Product Grunt is a curation of the worst new products, every day.

## Setup

```sh
git clone git@github.com:grant/productgrunt.git
cd productgrunt
npm install
npm start
```

## Dev setup

```sh
coffeegulp
```

### Built with

- :coffee: Coffeescript
- :cloud: Express
- :lipstick: Stylus
- :gem: Jade
- :tropical_fish: Gulp
